Gradient Animation Background

この背景デザインに必要なプロパティは以下の3つです。

body::afterにこれらのアニメーションを付与することで、PCでもスマ―トフォンでも、背景のグラデーションを表現できます。

  1. linear-gradient
  2. background-size
  3. background-position

linear-gradient

グラデーションを表現する値です。

linear-gradient(-60deg, red, blue)

のように指定すれば、

-60°方向に赤色から青色になるグラデーションを表現できます。

background-size

背景サイズの指定プロパティです。

background-size: 400% 200%;

のように指定すれば、

backgroundプロパティによる背景は縦4倍・横2倍のサイズとなります。

background-position

背景の表示位置のプロパティです。

background-position: 50% 100%

のように指定すれば、

backgroundプロパティによる背景の左上から横50%縦100%の部分が、その要素の横50%縦100%の部分に来るように表示されます。

実際のCSSコード

本ページ背景のCSSを記載しています。

            html,body{
               height: 100%
            }

            body::after{
               z-index: -1;
               position: fixed;
               top: 0;
               left: 0;
               width: 100%;
               height: 100%;
               content: "";
               background: #ee7752;
               background: -moz-linear-gradient(60deg, #ee7752, #e73c7e, #23a6d5, #23d5ab);
               background: -webkit-linear-gradient(120deg, #ee7752, #e73c7e, #23a6d5, #23d5ab);
               background: linear-gradient(-60deg, #ee7752, #e73c7e, #23a6d5, #23d5ab);
               background-size: 400% 400%;
               animation: gradient 15s ease-in-out infinite;
            }

            @keyframes gradient {
               0% {
               background-position: 50% 0%
               }
               50% {
               background-position: 50% 100%
               }
               100% {
               background-position: 50% 0%
               }
            }
